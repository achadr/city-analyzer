# City Analyzer

Minimal TypeScript skeleton for geographic analysis using Turf.js.

## Scripts

- `npm run dev` — run `src/index.ts` with ts-node
- `npm run build` — compile TypeScript to `dist/`
- `npm start` — run compiled app from `dist/index.js`
